# Curso-Python-UNC
Notebooks realizados para el curso virtual de python de la UNC

Consta de 3 tareas que buscaban reforzar los conocimientos aprendidos: 
1. Temperaturas notebook: Buscaba mostrar el conocimiento básico del uso de jupyternotebooks con markdowns y realizar un código sencillo de un transformador de grados celsius a grados farenheit.
2. Crear función: Buscaba crear una función básica con el argumento *args
3. Creación del inventario: Buscaba crear un gestionador sencillo de inventario, entrenando entonces el uso de diccionarios y listas.
